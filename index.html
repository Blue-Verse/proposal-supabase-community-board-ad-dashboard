프로젝트 소개

커뮤니티 게시판으로 유저 체류 시간을 높이고, 자체 광고 시스템으로 수익화 기반을 마련합니다. 백오피스는 상용 UI 템플릿으로 비용을 절감하고, 게시판 · 광고 트래킹 · 통계 핵심 기능에 예산을 집중하여 Supabase 기반으로 효율적으로 구축합니다.

  • 게시글(광고/공지/일반) HTML 에디터, 복수 이미지 업로드, 예약발행 등 게시 상태 관리, 신고 접수 · 처리
  • 광고 캠페인 생성 · 편집(기간/플랜/타겟/자동 만료)과 5종 광고 소재(피드 카드형 · 전면 팝업형 · 하단 배너형 · 바텀시트형 · 하단 슬라이드형) 관리
  • 광고 클릭 리다이렉트 라우팅 및 노출수 · 클릭수 · CTR 집계 그래프, CSV 내보내기
  • 광고주 전용 로그인 및 본인 캠페인 집행 현황 통계 · CSV
  • 사용자 게시판(웹뷰): 익명 닉네임 글 작성(500자 / 이미지 3장), 댓글 · 대댓글, 신고, 스포일러, 투표글, 좋아요 · 스크랩
  • 모바일 클라이언트 단 이미지 리사이징 · 압축, 클라이언트 AI 번역(캐시 · 로딩 · 재시도 · 원문보기, DB 미저장)
  • 게시글 SEO 메타정보(URL Slug, OG Image, JSON-LD)
  • 비기능: 트래킹 시 Supabase 과부하 방지 배치 최적화, AI 번역 API Key 노출 방지(Edge Functions 프록시)
  • Flutter 앱 웹뷰(WebView) 브릿지 연동

제안 포인트

1. 백오피스 비용 절감 · 핵심 기능 예산 집중

  • 관리자 · 광고주 대시보드는 상용 UI 템플릿(MUI 등) 커스터마이징으로 빠르게 세팅하여 디자인 · 퍼블리싱 비용을 최소화합니다.
  • 절감한 예산을 커뮤니티 게시판 · 광고 트래킹 · 통계 등 핵심 기능 구현에 집중 배분합니다.

2. 자체 광고 시스템 트래킹 · 통계

  • 5종 광고 소재 노출, 클릭 리다이렉트 라우팅, 노출수 · 클릭수 · CTR 집계 및 CSV 내보내기를 구축합니다.
  • 트래킹 이벤트는 버퍼링 · 배치 인서트와 집계 테이블 분리로 Supabase DB 부하를 최소화합니다.

3. Supabase 풀스택 · 보안

  • PostgreSQL Row Level Security 기반 4단계 권한 모델, Auth, Storage(이미지)를 안전하게 설계합니다.
  • AI 번역 API Key는 Edge Functions 프록시로 격리하여 클라이언트 노출을 차단합니다(번역 결과 DB 미저장).

4. 웹뷰 연동 & SEO

  • Flutter 앱 WebView 브릿지(JS ↔ Native) 통신으로 인증 · 이벤트를 연동합니다.
  • SSR 기반 SEO(URL Slug, OG Image, JSON-LD)로 게시글 검색 노출을 최적화합니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

앱 내 커뮤니티 기능을 강화하여 유저 체류 시간을 늘리고, 자체 광고 캠페인 운영을 통한 수익화 기반을 마련하는 것이 목표입니다. 관리자 · 광고주 백오피스는 상용 UI 템플릿을 적극 활용해 디자인 · 퍼블리싱 비용을 절감하고, 일반 유저가 사용하는 커뮤니티 게시판과 광고 트래킹 등 핵심 기능에 예산을 집중하여 Supabase 기반으로 효율적으로 구축합니다.

2. 사용자 정의

광고주광고 집행 주체로그인, 본인 캠페인 집행 현황 통계 조회, CSV 내보내기

3. 기능 요구사항

3.1 게시판 및 회원 관리자

  • [필수] 4단계 권한(관리자/광고주/유료/무료) 및 권한 신청 승인 · 반려
  • [필수] 전체 회원 목록 조회, 불량 유저 제재(차단)
  • [필수] 게시글(광고/공지/일반) HTML 에디터, 복수 이미지 업로드, 예약발행 등 게시 상태 관리
  • [필수] 게시글 · 댓글 목록 관리, 신고 내역 접수 · 처리

3.2 광고 캠페인 및 통계 관리

  • [필수] 캠페인 생성 · 편집(기간/플랜/타겟/상태/자동 만료)
  • [필수] 5종 광고 소재(피드 카드형/전면 팝업형/하단 배너형/바텀시트형/하단 슬라이드형)
  • [필수] 광고 클릭 시 외부 링크 리다이렉트 라우팅
  • [필수] 노출수/클릭수/CTR 집계 그래프, CSV 내보내기

3.3 광고주 대시보드

  • [필수] 광고주 로그인, 본인 캠페인 집행 현황 통계, CSV 내보내기

3.4 사용자 게시판 (웹뷰)

  • [필수] 관리자 작성 공지/광고 노출, 익명 닉네임 글 작성(500자, 이미지 3장)
  • [필수] 모바일 클라이언트 단 이미지 리사이징 · 압축
  • [필수] 댓글 · 대댓글, 악성 유저/게시글 신고
  • [필수] 스포일러(비로그인 시 일부 가림), 투표글
  • [필수] 좋아요 · 스크랩 · 내 글/스크랩 보기
  • [필수] 클라이언트 AI 번역(캐시/로딩/재시도/원문보기, DB 미저장)
  • [필수] SEO 메타정보(URL Slug, OG Image, JSON-LD)
